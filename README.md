# cb_cli
CloudBalkan Management Command Line Tool

https://www.cloudbalkan.com/

## Commands
### Servers

List all servers
> ./cli.sh servers list

View server details by id
> ./cli.sh servers view ID

Start|Stop|Restart server by id
> ./cli.sh servers restart ID

### Storage drives

List all storage drives
> ./cli.sh storage list

View storage drives details by id
> ./cli.sh storage view ID

### Domains

List all domains
> ./cli.sh domains list

View domains details by id
> ./cli.sh domains view ID
